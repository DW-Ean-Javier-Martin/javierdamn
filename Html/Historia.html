¿Qué es el HTML?
El HTML es un lenguaje de programacion para la elaboración de páginas web, basado en el uso de etiquetas encerradas por corchetes(<>) el cual se utiliza para traducir la estructura y la información en forma de texto, con la posibilidad de agregar objetos como imagenes, videos ETC. En la actualidad HTML es complementado por otras tecnologias como las las hojas de estilo(CSS) , el JavaScript entre otras.
Historia del HTML
Para empezar con la historia del HTML, daremos paso a sus origenes. Su origen fisico se remonta al físico Tim Berners-Lee, un trabajador del CERN(Organización Europea para la investigación Nuclear) quien a finales de 1989 propuso el lenguaje HTML junto con el protocolo HTTP, cuya finalidad era crear un medio para poder compartir información entre diversos “grandes” físicos de la época que trabajaban por todo el mundo.
